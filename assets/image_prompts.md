## Labeled Taxonomy Banner

```text
Use case: scientific-educational infographic-diagram
Asset type: GitHub README top banner, landscape 16:9
Primary request: Redraw a cute hand-drawn sketchnote style comparison diagram explaining cryptanalytic neural network extraction by analogy to block ciphers. The image MUST include readable labels inside the diagram.
Style/medium: clean hand-drawn marker-and-pencil sketchnote on warm off-white paper, friendly academic style, neat but slightly imperfect hand-drawn boxes and arrows.

Left panel content: input vector -> blue blocks labeled exactly "secret weights + bias" -> purple blocks labeled exactly "ReLU / pooling" -> output vector labeled exactly "logits / labels". Add small labels "chosen queries", "critical points", "boundary normals", and "recover weights, signs, geometry".
Right panel content: plaintext bits -> blue blocks labeled exactly "round keys" -> purple blocks labeled exactly "S-box / permutation" -> ciphertext bits. Add small labels "chosen plaintexts", "differential signals", and "recover key structure".
Color palette: soft blue for secret linear/key-dependent blocks, lavender for public nonlinear blocks, warm orange for arrows and keys, black handwritten text.
Constraints: all quoted text must be spelled exactly as given; no extra decorative text; no watermark; no logos; keep enough padding; make labels large and readable at GitHub README width.
```

## Style Notes

- For the top taxonomy banner, include short readable labels inside the image itself.
- Use Markdown labels only when they explain details that would make the image too crowded.
- Prefer warm cream paper, black marker outlines, pastel cyan/violet fills, and amber key/critical-point accents.
- Avoid dark cyberpunk visuals for this repository; the hand-drawn whiteboard style is friendlier and more memorable.
